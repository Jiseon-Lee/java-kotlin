글자 관련 CSS 속성

css에서 글꼴을 지정할 때는 font-family 속성을 이용하여 원하는 글꼴을 지정할 수 있다. 이때 글꼴명을 여러 개로 지정할 수 있는데 쉼표로 구분하며, 입력한 순서가 우선순위가 됨

글꼴을 여러 개로 지정하는 이유는 사용자의 시스템에 해당 글꼴이 없을 경우를 대비하기 위해 여러 개를 입력하여, 이 또한 없을 경우를 위해 마지막에 'serif', 'sans-serif', 'sursive', 'fantasy', 'monospace' 중 하나를 지정한다.


CSS에서의 색상 지정법

색상을 black, white 등 영문 색상명으로 지정가능

색상을 6자리의 16진수로 RGB 형식으로 지정가능

색상을 9자리의 10진수로 RGB 형식으로 지정가능

색상을 백분율의 RGB 형식으로 지정가능
