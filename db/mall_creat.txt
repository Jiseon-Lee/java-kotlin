회원관련 : 회원관리를 위한 테이블들
 - 테이블 : 회원정보, 주소, 포인트 사용내역
 - 프론트 : 로그인/아웃, 회원가입, 회원정보 수정, 회원탈퇴, 아이디/비밀번호 찾기, 포인트 내역보기, 주소록 관리
 - 어드민 : 회원검색, 정보보기, 상태값 변경(휴면, 탈퇴)

상품관련 : 상품관리를 위한 테이블들
 - 테이블 : 상품정보, 대분류, 소분류, 브랜드, 옵션별 재고
 - 프론트 : 상품목록 - 상품검색(분류, 브랜드별, 검색어, 기타 조건), 정렬(신상품, 인기상품, 가격대 등), 상품상세보기(수량, 옵션 선택)
 - 어드민 : 상품관리(등록, 수정), 상품이나 분류, 브랜드별 판매기록

주문관련 : 주문관리를 위한 테이블
 - 테이블 : 장바구니, 주문정보, 주문상세
 - 프론트 : 장바구니(등록, 수정, 삭제, 선택), 주문처리, 주문내역보기(주문취소, 주문완료)
 - 어드민 : 주문관리(상태값 변경, 주문검색)

CS관련 : QnA, FAQ
 - QnA테이블, FAQ테이블
 - 프론트 : 질문등록, 답변보기, FAQ활용(검색, 내용보기)
 - 어드민 : 질문검색(질문자, 답변여부), FAQ관리(등록, 수정, 삭제)

게시판관련 : 공지사항, 후기

 - 어드민 : 공지사항(글등록, 글수정, 글삭제), 후기(검색, 삭제, 포인트 회수)
